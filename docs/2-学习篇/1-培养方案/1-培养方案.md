# 培养方案简介

## 第一版培养方案（2022级~2024级）解读

### 学分构成
- 准出学分：150
    - 通识通修课程：61
        - 通识课程：14
            - 悦读：1
            - 科光：1
            - 美育：2
            - 劳育：2
                - 劳育课程学分：1
                - 劳育实践学分：1
                    - 这 1 学分需要完成基础劳动实践 10 小时，学科劳育实践 5 小时，综合劳动实践 5 小时方可获得。基础劳动实践建议大一就完成，苏州校区没有很多劳育岗位。
            - 人工智能通识课：1（2024级新开）
            - **另外在选课系统中选修课程类型为“通识”的课程学分：7（2024级）或8（2022级~2023级）**
        - 通修课程：47
            - 思政：17
            - 军事：4
            - 数学：14
            - 英语：8
            - 体育：4
    - 学科专业课程：55
        - 学科基础课程：40
        - 专业核心课程：15
    - 多元发展课程：28
        - 专业选修课程
            - 保研必修课：12
                :::tip 提示
                考虑保研的同学，这12学分务必修满。
                :::
            - 跨专业选修课程 + 公共选修课程：最低 16 学分。依个人情况而定，不保研的同学可以不修满保研必修学分而多修读跨专业选修课程和公共选修课程。最后成绩表中及格的，课程类型为“选修”的课程学分满 28 分，即到达毕业要求。
            :::tip 提示
            - 多修读的通识课程可以转换为多元发展学分。
            - 一些暑期项目可以转换为多元发展学分。
            - 五育可以有条件地转换为多元发展学分。
            - 大创可以转换为多元发展学分。
            - 公共选修课和通识课是两个不同的概念，提醒大家还要修 16 学分的选修课，不要被这些学分打乱生涯规划的节奏！
            :::
    - 毕业论文：6
:::tip 提示
从2021级学生开始，学生毕业前须在“思政选择性必修课”课程包中选修其中的一门课。如果学生已修读的课程中已包含该课程包中的任意一门课程，则可不用额外选修。具体课程包每年都会变动，请以当年课程包内容为准。
:::

            - 人文与社会科学：3（至少包含悦读 1 学分）
            - 自然科学与技术：3（至少包含科光 1 学分）
            - 美育：2
            - 劳育：2
                - 劳动教育课程：1

            - 思政理论课：20
            - 国家安全教育：1
            - 军事：4
            - 数学：14
            - 英语：8
            - 体育：4
            :::tip 提示
            2025级起，学校的体育课学分分配方法出现了一些变化，体育项目（如乒羽网）选修课从每门 1 学分变为每门 0.75 学分，而大三上学期和大四上学期的体质健康测试则按每次 0.5 学分计入学分绩。所以提醒大家，就算高年级没有体育课，也要注意锻炼身体！（就算不从功利的角度讲，身体是革命的本钱，平时要勤锻炼，保持健康哦！）
            :::

        - 专业选修课程
        - 跨专业选修课程
