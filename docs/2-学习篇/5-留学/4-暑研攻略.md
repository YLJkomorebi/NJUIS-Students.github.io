# 暑研攻略

暑研（暑期科研实习）一般安排在大三结束的暑假至大四上学期，前往海外教授的实验室进行较长期的科研实习。时长通常为 2–3 个月（短期）或 6 个月左右（长期）。如果能联系到满意的学校和教授，**建议尽量争取 6 个月的长期暑研**，这样更有可能拿到强推荐信或 return offer。

---

## 一、为什么要做暑研

1. **助力留学申请：** 对于想申请美国直博或 CS 强校热门硕士项目的同学来说，暑研是获取教授推荐信或 return offer 的核心途径。尤其对于申美的同学，美国学术界和高校普遍不太认可国内教授的推荐信，一封来自美国教授的强推往往具有决定性作用。
2. **提前适应海外环境：** 深入了解和适应美国的文化、生活和科研氛围，这些方面与国内存在不小的差异，提前体验有助于未来留学时更快融入。
3. **拓展研究视野：** 接触更前沿、更广阔的研究方向，有机会产出论文成果，同时对申请材料（尤其是 SoP 的撰写）也有很大帮助。

## 二、本校合作项目

南大提供美国暑研奖学金项目，同学们可以关注。以下是大致的时间线：

| 时间 | 事项 |
|------|------|
| 大三上 9–10 月初 | 开放申请 |
| 10 月末 | 资料提交 + 面试 |
| 11 月 | 教授录取反馈 |
| 12 月 – 次年 2/3 月 | 与教授自行联系，走对方学院流程，获取 DS-2019（相当于官方录取函） |
| 3–6 月 | 办理美签（J-1 签证），越早预约和面签越好，存在被拒或被 check 的可能。签证拿下后即可开始租房、办信用卡、订机票等 |
| 7 月 | 一般 7 月左右前往线下暑研 |

**一些 Tips：**

- 南大合作项目的教授大多为本校校友（在美做教职），申请难度相比自己套磁会小不少。也有部分清华本科背景的老师，要求会更高一些。
- 有的老师会额外安排面试或考核，有的则直接根据材料和自我介绍做出选择。
- 选择教授时务必仔细浏览其主页和近期论文。同时关注组内是否有本科生或实习生——一方面，有些教授不习惯指导本科生，缺乏相关经验；另一方面，如果组内实习生较多，竞争强推或 return offer 的压力也会更大。
