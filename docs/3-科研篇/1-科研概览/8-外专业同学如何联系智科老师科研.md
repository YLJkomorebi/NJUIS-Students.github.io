# 外专业同学如何联系智科老师科研

对于非智科院系的同学，不能直接参智科院系的本科生进组项目。但是如果希望参与智科老师的科研项目，也可以通过类似"套磁"的方式主动联系老师。整体思路与联系海外教授申请暑研类似，但由于是**校内科研**，联系方法、邮件重点以及准备材料会有所不同。

------------------------------------------------------------------------

## 一、

向智科方向的老师发送一封邮件，介绍自己的背景和能力，并询问是否可以加入老师的课题组参与本科生科研。

科研形式可能包括：

-   进入实验室做长期科研
-   参与某个具体项目
-   协助研究生完成项目或论文
-   参与大创项目或科研训练计划

与海外暑研不同的是，校内科研通常**不涉及签证、时间周期更灵活、沟通成本更低**，因此很多老师愿意接收本科生参与科研训练。

------------------------------------------------------------------------

## 二、如何寻找老师

### 1. 智科官网查找

可以直接进入 **智科相关学院官网**，在教师列表中查找老师信息。

通常可以获得：

-   教师邮箱
-   个人主页
-   研究方向
-   论文列表

如果老师提供了个人主页，可以进一步查看：

-   是否有实验室
-   是否有本科生参与科研
-   最近的研究方向和论文

------------------------------------------------------------------------

### 2. Google 搜索老师主页

有些老师在官网上的信息较少，可以通过 Google 搜索老师姓名找到主页。

搜索方法：

    老师名字拼音 + university

例如：

    jianguo wang

或

    jianguo wang nju

通常可以找到：

-   老师个人主页
-   实验室主页
-   Google Scholar 页面
-   论文列表

------------------------------------------------------------------------

### 3. 通过论文了解研究方向

在老师主页或 Google Scholar上查看老师最近几年的论文，了解研究方向是否与自己的兴趣一致。

如果邮件中能够提到老师的某些研究方向或论文，会更容易获得回复。

------------------------------------------------------------------------

## 三、联系老师的邮件内容

校内科研联系邮件通常包括以下内容：

### 1. 基本信息

简单介绍自己，例如：

-   来自哪个学院
-   什么专业
-   当前年级

------------------------------------------------------------------------

### 2. 学习成绩

由于是校内科研（一般这时候还没有科研经历），老师通常会比较关注**课程成绩和学习能力**，可以简单说明：

-   GPA 或排名
-   相关课程成绩（如机器学习、计算机视觉、算法等）

------------------------------------------------------------------------

### 3. 科研或项目经历

如果参与过以下内容，可以重点说明：

-   大创项目
-   课程项目
-   编程项目
-   科研训练

简要说明：

-   项目内容
-   自己负责的部分
-   使用的技术

------------------------------------------------------------------------

### 4. 获得的奖励或荣誉

例如：

-   奖学金
-   学科竞赛奖项
-   科研奖励

这些信息可以作为 **学习能力和执行能力的证明**。

------------------------------------------------------------------------

### 5. 英语能力

部分科研需要阅读英文论文，因此可以简单说明：

-   英语成绩（如六级、托福、雅思等）

------------------------------------------------------------------------

### 6. 附件材料

建议在邮件中附上：

-   个人简历（CV）
-   项目 demo 或 GitHub
-   项目报告或课程大作业

这些材料可以更直观地展示自己的能力。

------------------------------------------------------------------------

## 五、其他建议

-   邮件中尽量体现：

    -   学习能力
    -   编程能力
    -   阅读论文能力

-   如果有 **项目 demo / GitHub / 技术报告**，建议提供链接。

------------------------------------------------------------------------
